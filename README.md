# word2vec (NumPy)

This is a small project for the application fo jetbrains internship 2026.

The model is the **skip-gram variant with negative sampling**.
The goal was mainly to implement the the training step of word2vec model by just using numpy.
(forward pass, loss, gradients and parameter updates), not to build a
full-featured library.

## Running it

Install the only dependency:

```bash
pip install numpy
```

Then run:

python train.py
